Методы прогнозирования траектории движения фиксированной цели по имеющейся начальной траектории в двумерном пространстве

Описание данных и метрик

Для обучения и тестирования моделей был использован синтетически сгенерированный набор данных, имитирующий движение цели в двумерном пространстве. Датасет содержит 5000 траекторий, каждая из которых состоит из 40 последовательных временных шагов. Траектории генерируются программно с помощью функции generate_synthetic_trajectories. Каждая точка траектории описывается двумя координатами (x, y). Движение моделируется как комбинация квадратичного тренда и синусоидальных колебаний с добавлением гауссовского шума для имитации неточности измерений, что позволяет охватить широкий спектр нелинейных и стохастических паттернов, характерных для реальных задач слежения. Набор данных был разделен на обучающую и тестовую выборки в соотношении 4:1.

Для визуализации характера данных на (Рисунок 1)strong> представлены примеры сгенерированных траекторий.

Для оценки качества прогнозирования использовались следующие метрики:

ADE (Average Displacement Error)
- среднее евклидово расстояние между предсказанными и истинными позициями на всех временных шагах.
FDE (Final Displacement Error)
- евклидово расстояние между предсказанной и истинной позицией на последнем шаге прогноза.
MAE (Mean Absolute Error)
- средняя абсолютная ошибка по координатам x и y.

Методы прогнозирования и архитектура моделей

В работе рассматриваются два метода прогнозирования на основе рекуррентных нейронных сетей: с использованием сети типа LSTM (Long Short-Term Memory) [1][3] и сети типа GRU (Gated Recurrent Unit) [4][5]. Обе архитектуры относятся к классу RNN и специально разработаны для работы с последовательными данными, позволяя эффективно находить долгосрочные зависимости.

Архитектура моделей:

Обе модели имеют схожую структуру, различаясь лишь типом используемых рекуррентных слоев.

Входные данные: :

Трехмерный тензор размерности (batch_size, hist_len, feature_dim), где hist_len = 8 (длина последовательности (история)), feature_dim = 2 (координаты x, y).

Архитектура сети

Последовательная (Sequential) модель состоит из:

  1. Первый рекуррентный слой (LSTM или GRU) с 64 нейронами, возвращающий последовательности для последующих слоев.
  2. Слой Dropout с коэффициентом 0.2 для регуляризации.
  3. Второй рекуррентный слой (LSTM или GRU) с 128 нейронами, возвращающий последовательности.
  4. Слой Dropout с коэффициентом 0.2.
  5. Третий рекуррентный слой (LSTM или GRU) с 64 нейронами, возвращающий только последний выход.
  6. Полносвязный (Dense) слой с 32 нейронами и функцией активации ReLU.
  7. Выходной полносвязный слой без активации. Размерность выхода определяется как pred_len * feature_dim, где pred_len = 3 (длина прогнозируемой последовательности), следовательно, выходной слой имеет 6 нейронов.

Выходные данные:

Вектор размерности 6, который затем преобразуется в последовательность из 3 точек (x, y).

Гиперпараметры обучения

Для настройки процесса обучения обеих моделей (LSTM и GRU) были выбраны следующие гиперпараметры:

Оптимизатор
Adam (Adaptive Moment Estimation) [6].
Скорость обучения (Learning Rate)
0.001.
Параметры оптимизатора Adam
Использованы значения по умолчанию в TensorFlow/Keras :
  • beta_1 = 0.9 - коэффициент затухания для первого момента (среднего градиента);
  • beta_2 = 0.999 - коэффициент затухания для второго момента (нецентрированной дисперсии градиента);
  • epsilon = 1e-07 - малое число для численной стабильности.
Функция потерь (Loss Function)
Среднеквадратичная ошибка (Mean Squared Error, MSE).
Размер батча (Batch Size)
32.
Количество эпох (Epochs)
30.
Регуляризация Dropout
Во всех моделях после рекуррентных слоев применялся слой Dropout с вероятностью 0.2.
Инициализация весов
Использована инициализация по умолчанию в Keras (для плотных слоев - glorot_uniform, для рекуррентных - orthogonal).
